# 📉 Customer Churn Prediction App

This project predicts whether a customer will churn (leave the company) or stay.

## 🔥 Features
- User-friendly Streamlit Web App
- Predict churn using Machine Learning model
- Shows churn probability

## 🛠 Tech Stack
- Python
- Pandas
- Scikit-Learn
- Streamlit

## 🚀 How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run app:
   streamlit run app.py
